# JavaFX-Module-Chooser-GUI
This is a JavaFX-based desktop application that allows undergraduate computing students to create a profile and select their final-year modules based on their chosen course, following a clean Model–View–Controller (MVC) architecture.

# How to Run:
1. Download the Zip file and extract it.
2. Open the project in Eclipse or IntelliJ IDEA
3. Ensure JavaFX is correctly configured
4. Run the ApplicationLoader class
